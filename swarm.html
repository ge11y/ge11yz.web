What is this?

SWARM is an autonomous AI agent system. Two AI agents work together as a team — one supervises and plans, the other builds and executes. The human sets priorities, reviews results, and focuses on what matters.

It runs while you sleep. It builds while you work. You come home to finished work.

How the daily cycle works

🌅 Morning — Set Priorities

Human reviews overnight output, adds new tasks via Telegram. KORE breaks them into buildable pieces and assigns to CODY.

☀️ Day — Swarm Builds

CODY works autonomously on the MacBook Pro. Writes results to shared folder. KORE monitors progress and handles errors.

🌙 Evening — Review Complete Work

Human comes home to finished tasks. Reviews outputs in Telegram. Redirects, approves, or assigns follow-ups.

How to build your own swarm

You need two machines — one as supervisor, one as executor. Both connected to the internet. That's it.

1. Set up the supervisor (KORE)

  • Machine: always-on computer (Mac mini works great)
  • Install OpenClaw and configure your primary AI model
  • This machine manages memory, planning, routing, and Telegram communication

2. Set up the executor (CODY)

  • Machine: portable laptop (MacBook Pro)
  • Install Claude Code CLI
  • Configure bypassPermissions mode for autonomous operation

3. Connect them with Tailscale

  • Install Tailscale on both machines
  • Sign into the same Tailnet
  • They can now reach each other from anywhere on the internet

4. Enable passwordless SSH

  • Generate an SSH key on KORE
  • Add the public key to CODY's ~/.ssh/authorized_keys
  • Now KORE can call CODY without a password

5. Define your workflow

  • Task queue: KORE writes tasks, CODY picks them up
  • Result folder: CODY writes output, KORE reads and summarizes
  • Review loop: Human approves or redirects via Telegram
